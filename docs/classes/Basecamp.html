    PHP Class: Basecamp

    This class allows you to connect to the new Basecamp API. It is not compatible with the Basecamp Classic API. This class contains its own built-in RESTful service.

    The Basecamp class is fully cusomizable and easy to use!

    Initialization

    __construct(string $app_name, int $account_id, string $username, string $password) : void

    The Basecamp constructor doesn't require the arguments, you can set them afterwards.

    Here's an example of instantiation:

    required_once 'libraries/basecamp.php';

$basecamp = new Basecamp('My App', 123456, 'tjones', 'abc123');
